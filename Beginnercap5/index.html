practica de consola

Funciones de la consola

  • assert()(condition, message)
  • clear()(limpia consola)
  • error()(muestra un mensaje de error)
  • info()(muestra un mensaje de información)
  • log()(muestra un mensaje en la consola)
  • table()(muestra datos en forma de tabla)
  • warn()(muestra un mensaje de advertencia)
  • dir()(muestra el contenido de un objeto)

Se pueden utilizar estas funciones para depurar y probar el código en la consola del navegador.

funciones de tiempo,agrupacion,conteo

  • time()(inicia un temporizador)
  • timeLog()(muestra el tiempo transcurrido desde console.time())
  • timeEnd()(detiene el temporizador y muestra el tiempo transcurrido)
  • group()(inicia un grupo de mensajes en la consola)
  • groupEnd()(finaliza un grupo de mensajes en la consola)
  • count()(cuenta el número de veces que se ha llamado a esta función)
  • countReset()(reinicia el contador)
